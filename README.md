# 📊 HR Employee Attrition Analysis

## 🔍 Overview

This project analyzes employee attrition using an HR dataset to identify key factors influencing workforce turnover. Through data cleaning, statistical analysis, and visualization, the project uncovers patterns related to employee demographics, compensation, and departmental trends.

---

## 🎯 Objectives

* Analyze employee attrition patterns
* Identify factors contributing to employee turnover
* Compare attrition rates across departments and job roles
* Explore the relationship between salary, age, and attrition

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical computations
* **Seaborn & Matplotlib** – Data visualization

---

## 📁 Dataset

* IBM HR Employee Attrition Dataset
* Includes features such as:

  * Age
  * Department
  * Job Role
  * Monthly Income
  * Attrition (Yes/No)

---

## ⚙️ Project Workflow

### 1. Data Loading & Exploration

* Imported dataset using Pandas
* Inspected structure and column names
* Checked for missing values

### 2. Data Preprocessing

* Converted categorical attrition values into numerical format (Yes → 1, No → 0)

### 3. Exploratory Data Analysis (EDA)

* Calculated average age and income
* Analyzed attrition rate by department
* Evaluated average salary by job role

### 4. Data Visualization

* Attrition distribution (Count Plot)
* Age distribution (Histogram)
* Income vs Attrition (Box Plot)
* Correlation analysis (Heatmap)

---

## 📊 Key Insights

* Certain departments exhibit higher attrition rates
* Lower income levels are associated with increased attrition
* Workforce age distribution provides insight into employee demographics
* Correlation analysis highlights relationships between numerical features
